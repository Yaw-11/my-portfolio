# Personal Portfolio - Duodu Ignatius

A modern, responsive portfolio website showcasing your journey as an IT student and developer.

## 📁 Project Structure

```
my-portfolio/
├── index.html          # Main portfolio page
├── style.css           # All styling (modern & minimal design)
├── js/
│   └── main.js        # Interactive features & animations
├── images/            # Place your images here
├── style/             # Additional CSS files (if needed)
└── views/             # Additional pages (if needed)
```

## ✨ Features

✅ **Modern Minimal Design** - Clean, professional aesthetic  
✅ **Responsive Layout** - Works perfectly on mobile, tablet, and desktop  
✅ **Smooth Navigation** - Sticky navbar with smooth scrolling  
✅ **Hero Section** - Eye-catching introduction  
✅ **About Me** - Personal information and quick stats  
✅ **Skills Showcase** - HTML, CSS, JavaScript, Python  
✅ **Education Timeline** - Your academic journey  
✅ **Projects Section** - Showcase your work (Student Attendance Tracker & more)  
✅ **Contact Section** - Email and social media links  
✅ **Animations** - Smooth fade-in effects on scroll  

## 🚀 Getting Started

1. **Open the portfolio locally:**
   - Open `index.html` in your web browser
   - Or use VS Code Live Server extension for live preview

2. **Customize Your Portfolio:**
   - Edit the email and social media links in the Contact section
   - Add your GitHub, LinkedIn, Twitter, and Facebook profiles
   - Update project links when you complete projects
   - Add project images in the `images/` folder

3. **Add Your Projects:**
   - Once you complete the Student Attendance Tracker, update the project link
   - Add more projects as you build them
